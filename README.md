## Installation
Install hutlySite with npm
```bash
  npm install hutlySite
  cd hutlySite
```
    
## Run Locally
Clone the project
```bash
  git clone https://github.com/theibd56/hutlySite
```
Go to the project directory
```bash
  cd hutlySite
```
Install dependencies
```bash
  npm install
```
Start the server
```bash
  npm run dev
```
